# Quine-McCluskey-gui
GUI implementation of the Quine-McCluskey algorithm for minimizing Boolean functions. 

## Usage
Key features include:  
* Tabular entries  
* Minterm/Maxterm entries  
* Sum-of-Products outputs  
* Products-of-Sum outputs  
* Compliment output
